Digital Issue Management System

📌 About the Project

This project is designed to manage and track system-related issues reported in computer labs. Students and lecturers can report problems, and technicians can resolve them efficiently.

🎯 Objectives

  • Provide an easy way to report system issues
  • Allow technicians to track and resolve problems
  • Enable admin to monitor all complaints
  • Maintain a centralized database using Firebase

👥 Users of the System

  • Students – Submit system problems
  • Lecturers – Report lab-related issues
  • Technicians – View and update problem status
  • Admin – Monitor and manage all issues

⚙️ Technologies Used

  • HTML, CSS, JavaScript
  • Firebase Firestore Database
  • GitHub for hosting
  • VS Code for development

✨ Key Features

  • Online issue submission
  • Real-time database storage
  • Status tracking (Pending, In Progress, Resolved)
  • Admin control panel
  • Technician dashboard

🚀 Future Enhancements

  • User login authentication
  • Email notifications
  • Priority-based issue handling
  • Search and filter options
